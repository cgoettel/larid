# LarID

LarID is a phone and web app to help you identify gulls through feature-based filtering.

Gull identification is notoriously difficult because plumages vary significantly by age and season, with many overlapping characteristics between species. This app lets you select observable features (leg color, bill pattern, head color, etc.) to narrow down possible species and plumage combinations.
