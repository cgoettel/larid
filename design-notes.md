# Design

## Overview

The purpose of this app is to provide an easier way to identify gulls. Gulls have different plumages for the first few years of their lives, many of which overlap with other species' plumages.

## Usage

- My rough idea is that you can select all of the things you can see (e.g., eye ring color, leg color, head color, etc.) and then the app will filter down to which gulls it could be (e.g., adult male California Gull, first year plumage Arctic Tern, etc.).
- Should be able to use location to limit the likely gulls for the area.
